# toDoList

Lista de tarefas


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/antonio-cosmo/toDoList
```

Entre no diretório do projeto

```bash
  cd toDoList
```

Instale as dependências

```bash
  npm install
```

Inicie a aplicação no modo de desenvolvimento

```bash
  npm run dev
```


## Funcionalidades

- Criar tarefa
- Listar tarefas 
- Marcar como concluida a tarefa
- Deletar tarefa
